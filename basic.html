How to use style attribute :

we can use style attribute and add many functionalities to our webpage i.e: to change the font size to the whole para use font-size:30px and to change the font style use font-style as a value whereas style is a name so every attribute comes under the syntax of (name=value) so for style name the value have a particular syntax called (property:specification)

i.e: (font-size:30px) here px refers to pixels,(font-style:italic) after every value you should use (;) to differentiate the (property:specification).

there also exist another important value that is color which let the user to change the color here the colour of the color text is changed due to the style attribute with color value.
